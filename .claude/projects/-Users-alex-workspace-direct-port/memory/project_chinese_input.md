Типичный входной документ (examples/in_1.xlsx) содержит наименования товаров на китайском языке.
На этапе парсинга нужно переводить наименования на русский для корректной классификации по ТН ВЭД.

**Why:** TKS API ищет по русскоязычным описаниям. Без перевода классификация не работает.

**How to apply:** При работе с парсингом документов или classifier учитывать, что description может быть на любом языке и требует перевода.
